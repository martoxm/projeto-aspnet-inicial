# 📝 Exercício 01 — Formulário Simples (ASP.NET MVC)

Este exercício faz parte da minha prática com ASP.NET Core MVC.  
O objetivo é criar uma página de **cadastro simples**, enviando dados via formulário e exibindo o resultado em outra página.

---

## 🚀 Funcionalidades

- Formulário com campos:
  - Nome
  - Email
- Envio dos dados via método **POST**
- Exibição das informações enviadas pelo usuário
- Estrutura completa usando o padrão **MVC**


---

## ▶ Como Executar

1. Abra o terminal na pasta do projeto  
2. Execute:
*dotnet run*
3. Acesse no navegador:
[https://localhost:5001/Usuario/Cadastro (exemplo de link)]

---

## 🛠 Tecnologias Utilizadas

- ASP.NET Core MVC  
- C#  
- Razor Pages  
- HTML  
- .NET SDK  

---

## 📌 Objetivo do Exercício

- Criar um Model simples  
- Criar um Controller com métodos GET e POST  
- Criar Views para formulário e resultado  
- Entender o fluxo básico do padrão MVC
